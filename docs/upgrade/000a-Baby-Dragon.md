<SmallTitle>重要说明</SmallTitle>

如果飞龙宝宝附近 4.5 格无空军会触发狂暴，狂暴时秒伤为原来的 3 倍，攻速为 0.67 秒/次，即每次伤害为原来的 2 倍。
