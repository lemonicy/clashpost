<SmallTitle>重要说明</SmallTitle>

1. 驭水青龙是龙年佳节活动中的两个临时兵种之一，在活动的奖励之路中达到 3 000 红包即可解锁该兵种。
2. 驭水青龙使用水珠攻击，水珠会刺穿短距离的目标，造成直线的范围伤害。
3. 驭水青龙攻击时是三连发的，不过如果连发未结束时目标就被摧毁了，它可能会中断连发。
