<SmallTitle>重要说明</SmallTitle>

1. 愤怒水母的技能（洗脑，又称心灵控制）：让跟随的英雄在部署后的一段时间内优先攻击防御建筑。
2. 水母是飞行单位，可以对防御建筑进行远程攻击。
3. 水母处于跟随状态时不会成为目标或受到伤害，也不会触发陷阱。
4. 当水母跟随的英雄被击倒后，水母会变成普通部队，可以成为目标以及受到伤害，也可以触发陷阱了。
